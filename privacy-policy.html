Política de Privacidad

Esta es la política de privacidad para mi aplicación en desarrollo que utiliza la API de TikTok.

Esta aplicación está actualmente en desarrollo y se utiliza en un entorno local (localhost) para pruebas. No se recopila, almacena ni comparte información personal de los usuarios de forma permanente.

El único propósito de la integración con la API de TikTok es para probar la funcionalidad de visualización de métricas de mis propios videos.
